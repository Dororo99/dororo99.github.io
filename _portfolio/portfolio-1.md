This page collects representative projects from my CV. More detailed writeups can be split into separate portfolio entries later.

Fault-Tolerant Autonomous Driving Controller using Reinforcement Learning
======
- Project in the 2025 MATLAB AI Student Challenge.
- Configured a perception unit using radar and cameras.
- Developed an RL agent for collision avoidance during perception unit failures.
- Built a simulation environment using MATLAB and Simulink.

2D Grid-Based Autonomous Driving Algorithm
======
- Project in SMYD Club.
- Applied path planning algorithms including A*, Hybrid A*, and RRT*.
- Implemented PID control for vehicle dynamics.

3D Gaussian Splatting Reconstruction
======
- Graduation project reconstructing objects from different backgrounds in 3D space.
- Built a custom dataset from self-captured videos.
- Extracted object-specific point clouds using COLMAP, Gaussian Grouping, and SAM.
- Improved Gaussian mixture model performance using a clustering algorithm.

Computer Vision Measurement
======
- Project in the 2023 CJ Logistics Future Technology Challenge.
- Measured the volume of products at rest and in motion on a conveyor belt using a smartphone.
- Used image processing and computer vision techniques to estimate length, width, and height.
