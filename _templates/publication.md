# Publication Entry Template

Copy the front matter below into a new file under `_publications/`.

Use a filename like `_publications/2026-05-19-short-title.md`.

```yaml
---
title: "Paper Title"
collection: publications
category: manuscripts
permalink: /publication/2026-05-19-short-title
excerpt: "One-sentence summary of the paper."
date: 2026-05-19
venue: "Journal or Conference Name"
paperurl:
citation: "Author1, Author2, and DoHyun Lim. &quot;Paper Title.&quot; Journal or Conference Name, 2026."
---
```

Set `category` to one of:

- `preprints`
- `manuscripts`
- `conferences`
- `posters`

Add the abstract, notes, code link, PDF link, or project link below the front matter.
